# Math Learning Platform

## Краткое описание
Система для обучения математике с интерактивными задачами и тестами. Проект предназначен для учителей, учеников и самостоятельного изучения.

## Основные функции
- Решение заданий в реальном времени
- Ведение статистики прогресса
- Различные типы задач и уровней
- Требуется дополнить описание функционала

## Установка и запуск
1. Клонирование:
```bash
git clone https://github.com/raymaksot/math-learning-platform.git
cd math-learning-platform
```
2. Установка зависимостей:
```bash
pip install -r requirements.txt
# или, если используется Node.js для TypeScript:
npm install
```
3. Запуск платформы:
```bash
# Для Python
python main.py

# Для TypeScript/Node.js
npm start
```
> Уточните точную команду запуска для вашего проекта.
